concepCy
====================================

.. image:: https://img.shields.io/github/stars/JulesBelveze/concepcy.svg?style=social&label=Star&maxAge=2592000
   :target: https://github.com/JulesBelveze/concepcy

concepCy is a spaCy wrapper for `ConceptNet <https://conceptnet.io/>`_, a freely-available semantic network designed to help computers understand the meaning of words.
It allows you to query `ConceptNet.io <https://conceptnet.io/>`_ to extract word meanings directly from the resource itself.


Content
---------------------------------

- **Getting started**: installation instructions and step-by-step guides on how to use concepCy.
- **ConceptNet**: the main concepts (no pun) behind ConceptNet.
- **API references**: docs for objects and functions.
